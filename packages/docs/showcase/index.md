# Showcase

Projects building with DappQL in production, and case studies of what that looks like.

## Case studies

### [Ripe Finance](/showcase/ripe-finance)

*5 engineers. 200 contracts. The library that fell out.* The origin story of DappQL, as told through the project that forced it into existence. Weekly ABI churn, a viral testnet day that didn't melt the RPC bill, and why typed contract context matters for AI agents too.

### [Underscore Finance](/agents/case-studies/underscore)

*75 contracts on Base. Live-agent DeFi analyst sessions.* How Claude Code connected to `@dappql/mcp` reasons about a full DeFi protocol: DeFi math from typed reads, write-safety catches, APY derivation from historical blocks.

## Built with DappQL

- [Underscore Finance](https://underscore.finance) — AI-agent wallet infrastructure, ships DappQL-generated SDKs.
